PNG

  • This is a Portable Network Graphic file.
  • Though it is interchangeable with a GIF file, it does not support animations.
  • This was created to provide a full image with out copyright limits.
  • PNG also has transparency options.

    GIF

  • This is a Graphic Interchange File.
  • It suports simple animations.
  • This only is compatible with 256 colors.

    JPEG

  • This is a Joint Photographic Experts Group.
  • It has millions of colors availible to use because it can blend colors.
  • This is good to use for still images or photographs.
